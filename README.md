# 🌌 Universo del Conocimiento

**Plataforma educativa gamificada** para niños de 10–12 años.  
Convierte el refuerzo escolar en un viaje espacial con planetas, desafíos y recompensas.

---

## 🚀 Stack

| Capa | Tecnología |
|------|-----------|
| **Frontend** | Next.js 16 + TypeScript + Tailwind CSS v4 + Framer Motion |
| **Estado** | Zustand + localStorage |

## 📁 Estructura

```
web/          → Frontend Next.js
shared/       → Tipos TypeScript compartidos
docs/         → Documentación académica
```

## 📦 Funcionalidades

- 🌍 **3 planetas temáticos**: Numérix (Matemáticas), Letralia (Lenguas), Naturae (Ciencias)
- 🎮 **Dashboard interactivo** con planetas, cursos y niveles
- 🪐 **Planetas 3D animados** con textura, anillos y elementos temáticos
- 🌌 **Fondo cósmico interactivo** con estrellas y nebulosas (Canvas optimizado)

## 🏃 Cómo empezar

```bash
cd web
npm install
npm run dev     # → http://localhost:3000
```
