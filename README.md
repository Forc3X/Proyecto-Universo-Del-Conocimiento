# 🌌 Universo del Conocimiento

**Plataforma educativa gamificada** para niños de 10–12 años.  
Convierte el refuerzo escolar en un viaje espacial con planetas, desafíos y recompensas.

> **Tesis de Grado** — PNF Informática, Trayecto II  
> UPTAEB "Federico Brito Figueroa" (La Victoria, Aragua, Venezuela)

---

## 🚀 Stack

| Capa | Tecnología |
|------|-----------|
| **Frontend** | Next.js 16 + TypeScript + Tailwind CSS v4 + Framer Motion |
| **Estado** | Zustand + localStorage |
| **Backend** | Express 5 + TypeScript + MySQL |
| **IA adaptativa** | Ollama + Gemma (próximamente) |

## 📁 Estructura

```
web/          → Frontend Next.js
server/       → Backend Express + MySQL
shared/       → Tipos TypeScript compartidos
docs/         → Documentación académica
```

## 🧭 Fases del proyecto

Ver [`PLAN.json`](./PLAN.json) para el plan detallado o [`CHANGELOG.md`](./CHANGELOG.md) para el historial de cambios.

## 🏃 Cómo empezar

```bash
# Frontend
cd web
npm install
npm run dev     # → http://localhost:3000

# Backend (cuando esté listo)
cd server
npm install
npm run dev     # → http://localhost:4001
```
